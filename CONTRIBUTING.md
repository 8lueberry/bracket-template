# Contributing

## Machine setup

- Install `nvm`

## Getting started

- Clone project `https://github.com/danlevan/bracket`
- Go to the project repo
- Run `nvm use`
- Run `npm install`
- Test the installation by running `npm test`

## Repo structure

|-dist
|-examples
|-scripts
|-tests
|-index.js

## Testing

- `$ npm test`

## Distributing

- `$ npm run build` will minify in `/dist/bracket.min.js`
